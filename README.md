# base

A super bare bones, nothing to it portfolio. 

## Project setup

Clone and install
```
cd ~/Sites
git clone git@github.com:etuchscherer/portfolio.git
cd ~/Sites/portfolio
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Deploy
```
yarn run deploy
```
